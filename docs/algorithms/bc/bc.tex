\section{\textit{Betweenness Centrality}}

O algoritmo de \textit{Betweeness Centrality} é muito utilizado para o estudo de redes sociais devido a indicar a medida de centralidade para cada vértice da rede a que foi aplicado,isto é, calculará para cada vértice um grau de importância/influência. 

O grau de importância de vértice é dado pelo número de caminhos mais curtos que passam nesse mesmo vértice.
